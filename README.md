# 🌊 Fluid

🚀 A **tiny but agile** microservice framework built in **Java 24** with first-class support for **Docker 🐳**, **Kubernetes ☸️**, and **Kafka 📨** event streaming.  
Built for **speed, scale, and simplicity**.
